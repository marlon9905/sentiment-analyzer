# Sentiment Analyzer

Proyecto académico para el consumo de servicios de Inteligencia Artificial en la nube.

## Descripción
Aplicación web que analiza el sentimiento de un texto utilizando modelos de NLP
a través de la API de Hugging Face.

## Estructura
- `/backend`: API en Node.js + Express
- `/frontend`: HTML, CSS y JavaScript

## Instalación
```bash
cd backend
npm install
npm start
